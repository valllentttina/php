# Контрольные вопросы

### 1. Какие способы установки PHP существуют?

* Установка отдельного PHP с официального сайта
* Установка в составе сборок (XAMPP, OpenServer, WAMP, MAMP)
* Установка через пакетные менеджеры (apt, yum, brew и др.)
* Использование Docker-контейнеров

### 2. Как проверить, что PHP установлен и работает?

1. Создать файл 
2. Открыть файл в браузере
3. Если отображается информация значит PHP работает

### 3. Чем отличается оператор `echo` от `print`?

* `echo` может выводить несколько строк через запятую
* `print` возвращает значение `1`, поэтому может использоваться в выражениях
* `echo` работает немного быстрее
* `print` принимает только один аргумент
